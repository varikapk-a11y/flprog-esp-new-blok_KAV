# flprog-esp-new-blok_KAV
Пользовательский блок ESP8266 для FLProg. Добавление входов/выходов.

Блок `esp_new_8266` для расширенного управления через FLProg.

## Текущая задача
Расширение функционала: добавление дополнительных цифровых и аналоговых входов/выходов.

## Структура проекта
*   `esp_new_8266.ubi` — основной файл блока для FLProg
*   `documentation/` — папка для схем и описаний (можно создать позже)

## Как использовать
1.  Скачайте файл `txt (сменить расширение на .ubi)
2.  Импортируйте его в библиотеку пользовательских блоков FLProg.
